# 📊 Tech Layoffs Data Analysis Pipeline (2020–2023)

This project demonstrates an end-to-end data analysis workflow using SQL, starting from raw data cleaning to exploratory data analysis (EDA) on a dataset of global tech layoffs (2020–2023). The objective was to improve data quality, uncover trends, and generate actionable insights.

---

## 🔧 Project Workflow  

### 1. Data Cleaning & Preprocessing  
- Identified and removed duplicate records using SQL window functions  
- Standardized inconsistent data formats (text, dates, numeric fields)  
- Handled missing/null values using filtering and imputation techniques  
- Optimized dataset by removing irrelevant columns to improve performance  

---

### 2. Exploratory Data Analysis (EDA)  
- Performed SQL-based analysis to identify trends and patterns  
- Analyzed layoffs across companies, industries, countries, and funding stages  
- Detected outliers such as unusually high layoff percentages  
- Generated insights to support data-driven decision-making  

---

## 🛠 Tech Stack  
- SQL (MySQL)  
- MySQL Workbench / Command Line
